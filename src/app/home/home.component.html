อุปกรณ์ IOT ที่ใช้ทำชิ้นงาน

ตู้จำหน่ายสินค้าระบบอัตโนมัติ

ESP32

ESP32 คือ wifi microcontroller ที่ถูกพัฒนาต่อจาก ESP8266 โดยเพิ่ม CPU เป็น 2 core, Wi-Fi ที่เร็วขึ้น, มีขา GPIO ให้ใช้งานมากขึ้น และรองรับ Bluetooth อีกด้วย นอกจากนี้ ESP32 ยังมาพร้อมกับ touch-sensitive pins ที่สามารถใช้ปลุก ESP32 จากโหมด deep sleep และยังมี hall effect sensor และ temperature sensor ในตัว (รุ่นล่าสุดของ ESP32 ไม่มี temperature sensor ในตัวอีกแล้ว)

LCD

LCD ย่อมาจาก Liquid Crystal Display ซึ่งเป็นจอแสดงผลแบบ (Digital ) โดยภาพที่ปรากฏขึ้นเกิดจากแสงที่ถูกปล่อยออกมาจากหลอดไฟ ด้านหลังของจอภาพ (Black Light) ผ่านชั้นกรองแสง (Polarized filter) แล้ววิ่งไปยัง คริสตัลเหลวที่เรียงตัวด้วยกัน 3 เซลล์คือ แสงสีแดง แสงสีเขียวและแสงสีนํ้าเงิน กลายเป็นพิกเ:ซล (Pixel) ที่สว่างสดใสเกิดขึ้น

Load Cells

โหลดเซลล์ (Load cell) คือ เซนเซอร์ที่สามารถแปลงค่าแรงกด หรือแรงดึง เป็นสัญญาณทางไฟฟ้าได้ เหมาะสำหรับการทดสอบคุณสมบัติทางกลของชิ้นงานโหลดเซลล์ถูกนำไปใช้ในอุตสาหกรรมหลากหลายประเภท

Counter Sensor

Counter Sensor เป็นเซนเซอร์นับจำนวน ให้สัญญาณ Digital และ Analog ออกมา เมื่อมีวัตถุมาบังที่ร่องตรวจนับ (มีขนาดกว้าง 5 มิลลิเมตร) พร้อมหลอดไฟ LED แสดงการทำงาน สามารถนำไปนับจำนวนเหรีญหรือวัดความเร็วรอบมอเตอร์ได้

Switch

ปุ่มกด (Push Button Switch) เป็นสวิตช์ที่ใช้สำหรับเปิด-ปิดวงจรไฟฟ้าแบบชั่วคราว เมื่อกดปุ่ม วงจรจะถูกเชื่อมต่อ และเมื่อปล่อยปุ่ม วงจรจะถูกตัดการเชื่อมต่อ ปุ่มกดนี้มักใช้ในโครงการอิเล็กทรอนิกส์และหุ่นยนต์ต่างๆ โดยเฉพาะกับบอร์ดไมโครคอนโทรลเลอร์ เช่น Arduino เพื่อให้ผู้ใช้สามารถควบคุมการทำงานของโปรแกรมหรืออุปกรณ์ต่างๆ โดยการกดปุ่มเพื่อสั่งให้วงจรทำงานหรือหยุดทำงาน
